Value Type:
1.Store the actual data(ex:int,float,char)
2.Allocated On Stack
3.When assigned to another variable or passed as method parameter, a copy of the value is created.

Referance Types:
1.Store a reference(Memory Address) to the location where the data is stored.(Ex:Classes,Arrays,Interfaces & Delegates)
2.Allocated on heap
3.When assigned to another variable or passed as method parameter the reference is paassed not actaul data.
